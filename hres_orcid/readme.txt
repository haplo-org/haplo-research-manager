title: ORCID
--

This plugin defines the @hres:orcid@ text type and provides the infrastructure for integrating with ORCID services.


h2. Features

h3(feature). "hres:orcid"

h3(property). create(string)

Creates an ORCID (using the @hres:orcid@ text type) using the supplied string.

h3(property). url(orcid)

Returns the external url associated with this ORCID.

h3(property). asString(orcid)

Returns a string representation of an ORCID
